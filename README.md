# 🛒 Kabadibajar - MERN Stack Project

## 📌 Description

Kabadibajar is a MERN stack application where users can buy/sell scrap items online.

## 🚀 Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/codewithnitesh115/kabadibajar-mern.git
```

### 2. Install dependencies

#### For backend:

```bash
cd server
npm install
```

#### For frontend:

```bash
cd client
npm install
```

### 3. Setup environment variables

Create `.env` file in server folder:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret
```

### 4. Run the app

#### Backend:

```bash
npm run dev
or
node server.js
```

#### Frontend:

```bash
npm start
```

## 📸 Features

* User authentication
* Product listing
* Buy/Sell items
* REST API integration
